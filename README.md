# 自制编程语言——xlang

类似与python + cpp语法的x语言，x语言解释器命名为xlang。



## 目前功能支持：

1. 注释：//、/*	*/
2. 变量类型：int、float、string、list
3. 变量定义、输出
4. 运算符号：+、-、*、/、%
5. 比较运算符：==、!=、>、<、>=、<=
6. 逻辑运算符：&&、||
7. 赋值运算符：=、+=、-=、*=、/=、%=
8. 左++、右++、左--、右--
9. 条件控制：if...else...
10. 循环控制：for、while、do...while、break、continue
11. 作用域：{}
12. 函数：func定义、调用、return
